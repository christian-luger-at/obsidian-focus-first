# What is Focus First?

Stop guessing what to work on next. **Focus First** sorts your Obsidian tasks into the **Eisenhower matrix**, by urgency (due date) and importance (priority), so the next right action is always obvious.

![The Focus First view](/screens/tour.gif)

It reads standard checkbox tasks from your vault (compatible with the [Tasks plugin](https://obsidian.tasks.org/) format - due dates, priorities, tags) and places each one automatically:

|               | Urgent                              | Not urgent                          |
| ------------- | ----------------------------------- | ----------------------------------- |
| **Important** | **Do** - handle now                 | **Schedule** - plan for later       |
| **Not important** | **Delegate** - hand off if possible | **Eliminate** - reconsider or drop |

No manual sorting needed - though you can still pin any task to a quadrant by hand.

## Feature overview

Everything Focus First does, with links to where it's explained.

**Classification & matrices**

- **Automatic classification** into quadrants by urgency (due date) and importance (priority) - no setup beyond your existing tasks. → [How tasks are classified](/guide/classification)
- **Two matrices, one view** - switch between Eisenhower and Value/Effort from the header. → [The two matrices](/guide/matrices)
- **Manual quadrant tags** - pin any task to a quadrant with a tag, overriding the automatic logic. → [Overriding classification](/guide/classification#overriding-the-automatic-classification)
- **Drag & drop** a task between quadrants to re-classify it instantly. → [The two matrices](/guide/matrices)

**Focus & planning**

- **Focus / today plan** - an ordered, numbered `#focus` shortlist with drag-to-reorder and an optional daily target, built for Eat the Frog / Ivy Lee / MITs. → [Focus, size & filters](/guide/focus-size-filters#focus-tasks-today-plan)
- **Task size & quick wins** - tag tasks `#s`/`#m`/`#l` and filter to short ones. → [Task size](/guide/focus-size-filters#task-size)
- **Hide & snooze** - remove a task from the matrix, or hide it until a date. → [Hide & snooze](/guide/focus-size-filters#hide-snooze-tasks)
- **Future tasks** - show, dim, or hide tasks that aren't actionable yet. → [Future tasks](/guide/focus-size-filters#future-tasks)

**Working with the list**

- **Details on demand** - a clean list of titles; hover (desktop) or tap (mobile) a task for its "why here" reason, metadata, and one-click actions (complete, focus, hide, postpone, priority, size), each with a brief **Undo**. → [How tasks are classified](/guide/classification)
- **Search & filters** - search text, or filter by due-date bucket or task size. → [Search & filters](/guide/focus-size-filters#search-filters)
- **Grouping & sorting** - group and sort within each quadrant, configurable per quadrant. → [Settings](/guide/settings#grouping-sorting)

**Capture, scope & embedding**

- **Quick add** - capture a task without leaving the view, into a configurable inbox note. → [Adding tasks quickly](/guide/settings#adding-tasks-quickly)
- **Folder scope** - scan the whole vault or a single folder. → [Settings](/guide/settings)
- **Embed task lists** in any note via a `focus-first-tasks` code block. → [Embedding tasks in a note](/guide/embedding)

**Appearance & platform**

- **Adjustable font size**, **auto-refresh** on file changes, and **locale-aware dates**.
- Works on **desktop and mobile**, in **English and German**.
